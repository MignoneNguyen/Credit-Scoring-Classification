# Credit-Scoring-Classification
Credit scoring is an essential process used by financial institutions to evaluate how likely an individual or business is to repay borrowed money. This project aims to develop classification models that predict credit scores by analyzing factors like income, outstanding debt, and past credit behavior.

# Dataset Description
Link: https://www.kaggle.com/datasets/parisrohan/credit-score-classification

This project uses the Kaggle dataset: Credit Score Classification by Rohan Paris. The dataset simulates real-world financial data collected by a global finance company to assess customer creditworthiness.

It contains detailed information about individuals, including demographic data, banking details, and credit behavior. Key features include income, number of bank accounts, credit cards, outstanding debt, credit utilization ratio, payment behavior, and credit history.

The dataset is split into training and testing sets, with the training set containing around 100,000 records and 28 features, including the target variable credit_score, which categorizes customers into different credit risk levels (e.g., Poor, Standard, Good).
